-- Backup and Restore

This folder contains scripts and documentation demonstrating
a practical database backup and restore strategy.

-- Files

- Backup script.sh  
  Automates full database backups.

- Restore script.sh  
  Restores database from a backup file.

- Backup strategy.md  
  Documents backup scheduling, retention, and recovery approach.

-- How to Use :

1. Make scripts executable:
   chmod +x \*.sh

2. Run backup:
   ./Backup\ script.sh

3. Restore database:
   ./Restore\ script.sh backup_file.sql

- Disaster recovery planning
